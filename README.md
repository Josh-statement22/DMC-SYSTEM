## ⚡INSTALLATION GUIDE

Follow these steps to set up the DMC-ERP project locally.
---
### 1️⃣ Clone the Repository
```bash
git clone <your-repo-url>
cd DMC-ERP
```
---
### 2️⃣ Install PHP Dependencies
```bash
composer install
```
---
### 3️⃣ Install Node.js Dependencies
```bash
npm install
```

---
### 3️⃣ Install Node.js Dependencies
```bash
npm install
```
---
### 4️⃣ Set Up Environment
```bash
cp .env.example .env
php artisan key:generate
```

---
### 5️⃣ Build TailwindCSS (open another terminal for it)
```bash
npx tailwindcss -i resources/css/app.css -o public/css/app.css --watch
```

---
### 6️⃣ Run Laravel Server
```bash
php artisan serve
```
